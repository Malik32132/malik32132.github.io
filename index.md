1. Locate the downloaded file in your `Downloads` folder.

### For macOS Users

1. Open your `Downloads` folder.
2. Find the downloaded `.dmg` file and double-click it.
3. Drag the Sosina application icon into your Applications folder.
4. Open your Applications folder and double-click on the Sosina app to launch it.

2. Use the command `cd Downloads` to navigate to your `Downloads` folder.
3. Use the command `sudo dpkg -i sosina-installer-linux.deb` (replace with the correct filename if necessary).
4. After installation, run `sosina` from your terminal to start the application.
